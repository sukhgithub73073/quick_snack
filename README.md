# Easy Toast

A simple toast/snackbar utility for Flutter using GetX.

## 🚀 Features

- ✅ Success toast (green)
- ❌ Failure toast (red)
- ⚠️ Warning toast (yellow)
- 🎨 Custom color toast

## 📦 Installation

Add to your `pubspec.yaml`:

```yaml
dependencies:
  quick_snack:
    git:
      url: https://github.com/sukhgithub73073/quick_snack.git
